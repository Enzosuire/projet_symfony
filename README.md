# Projets Symfony

Applications web développées avec :

- Symfony 6
- PHP 8
- Doctrine ORM
- Twig
- MySQL

## Projets réalisés

- **Blog** – gestion d’articles et back-office  
- **Cuisine** – gestion de recettes  
- **Auto Retro Club** – présentation véhicules  
- **Post Manager** – gestion de publications  
- **Produits High-Tech** – mini e-commerce  

## Fonctionnalités

- CRUD complet  
- Authentification  
- Back Office  

## Stack technique

Symfony / PHP / MVC
